## K-Means Fitness Clustering

### Objective

To analyze synthetic fitness tracker data using K-Means clustering, identifying patterns in user activity and sleep behavior.

### Key Insights

* Distinct clusters emerge based on activity levels (active, moderate, inactive)
* Sleep duration correlates with activity intensity
* K-Means effectively groups unlabeled behavioral data
* Visualization helps interpret cluster separation
* Synthetic data allows controlled experimentation

### Concepts Demonstrated

* Unsupervised Machine Learning
* K-Means Clustering
* Data Visualization (Matplotlib)
* NumPy-based data generation

## Gaussian Clustering Simulation

### Objective

To generate synthetic Gaussian-distributed datasets and evaluate clustering accuracy using K-Means.

### Key Insights

* Cluster separation distance affects classification accuracy
* Gaussian noise introduces realistic variability
* K-Means performs well on clearly separable data
* Label alignment is required for correct accuracy evaluation
* Demonstrates limitations of clustering with overlapping data

### Concepts Demonstrated

* Synthetic data generation
* Probability distributions (Gaussian)
* Clustering accuracy evaluation
* NumPy and Scikit-learn usage

## Star Schema Visualization

### Objective

To model and visualize a data warehouse star schema using graph-based representation.

### Key Insights

* Star schema simplifies complex relational data structures
* Fact tables connect to multiple dimension tables
* Visualization improves understanding of database design
* Graph structures clearly represent relationships
* Useful for data warehousing and analytics systems

### Concepts Demonstrated

* Data warehouse design
* Graph modeling (NetworkX)
* Data relationships and normalization
* Visualization techniques
