# Java Projects


# Card Game Simulation
## Objective
To implement a simple card game using object-oriented programming principles, simulating deck creation, shuffling, and player interaction.
## Key Insights
	• Encapsulation allows clean separation between Card, Deck, and Player logic
	• Randomized shuffling ensures fairness in gameplay
	• Game logic demonstrates comparison-based decision making
	• Use of collections simplifies deck management
	• The design is easily extendable for more complex card games
## Concepts Demonstrated
	• Classes and objects
	• ArrayList usage
	• Randomization techniques


# Airline Reservation System
## Objective
To design a flight reservation system using a custom-built linked list to manage passenger records and ensure data integrity.
## Key Insights
	• Custom linked list implementation demonstrates low-level data structure understanding
	• Duplicate passengers are effectively handled using hashing techniques
	• The system supports dynamic insertion and deletion of records
	• Encapsulation ensures modular and maintainable design
	• Highlights trade-offs between built-in and custom data structures
## Concepts Demonstrated
	• Linked list implementation from scratch
	• Object-oriented design
	• HashSet for duplicate detection
	• Data management operations (CRUD)

# Hangman Game
## Objective
To develop an interactive console-based Hangman game that reinforces string manipulation and user input handling.
## Key Insights
	• Random word selection ensures replayability
	• Tracking guessed letters prevents redundant input
	• Game logic enforces win/lose conditions effectively
	• Demonstrates control flow and loop management
	• Highlights importance of user interaction design
## Concepts Demonstrated
	• Arrays and strings
	• Loops and conditionals
	• HashSet for tracking used inputs

# Request Management System
## Objective
To build a simple request tracking system with file persistence, allowing users to store and retrieve structured data.
## Key Insights
	• File handling enables persistent storage beyond runtime
	• Data serialization using delimiters allows easy reconstruction of objects
	• CLI-based interface demonstrates user interaction design
	• System supports basic CRUD operations
	• Highlights importance of error handling in I/O operations
## Concepts Demonstrated
	• File I/O in Java
	• ArrayList data management
	• Exception handling
